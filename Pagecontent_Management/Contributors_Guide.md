### I. Workshop hochladen
1.) Erstelle einen neune Ordner mit deinem Workshopnamen in diesem Order: [/Pagecontent_Management/Workshops](/Pagecontent_Management/Workshops)

2.) Erstelle in deinem neuen Ordner eine README.md Datei, in welcher du die Element deines Workshops kurz beschreibst

3.) Lade in deinem Ordner die Workshop-Unterlagen, wie beispielsweise der Handreichung für Mentor:innen oder Anleitungen für Teilnehmende. 

4.) Benenne diese aussagekräftig und verlinke sie über Hyperlinks in deinem README.md. In GitHub geht das wie folgt: [Name-Hyperlink] und direkt anschließend (Pfad-zur-Datei/Dateiname)

5.) Erstelle einen Pull Request. Wenn alles passt wird der Workshop von unseren Ehrenamtlichen in das Repository gemergt.

### II. Rahmenbedingungen Workshop-Erstellung
1.) Unser [Leitfaden zur Workshop-Erstellung](/Material/Leitfaden-Workshop-Erstellung_V1.pdf) hilft dir bei der Erstellung von Workshops weiter

2.) Benutze zum Erstellen von Anleitungen für Teilnehmende dieses [Template](/Material/Vorlage_Teilnehmenden_Anleitung.af). Für die Handreichung für die Mentor:innen steht diese [Vorlage](/Material/Vorlage_Mentorinnen_Anleitung.af) zur Verfügung.
