# O-Bi-Bo
Die Offene Bildungsbox (O-Bi-Bo) ist ein Workshop-Austausch-Format, das die Erstellung und Durchführung von (MINKT-)Workshops erleichtern soll.
- Der [Leitfaden zur Workshop-Erstellung](https://github.com/offenebildungsbox/O-Bi-Bo/tree/main/Pagecontent_Management/Material) unterstützt Ehrenamtliche bei der Themenfindung, Ablaufplanung und dabei, Material für Mentor:innen  und Teilnehmende vorzubereiten.
